# spring-hateoas-sample
Spring Boot and Spring HATEOAS sample

# Endpoint

| エンドポイント | メソッド | 用途 |
|:-------------:|:------------|:------------|
| /users | GET | ユーザー一覧情報取得  |
| /users/{id} | GET | ユーザー情報取得  |
| /users | POST | ユーザー情報登録 |
| /user/{id} | PUT| ユーザー情報登録 |
| /users/{id} | DELETE | ユーザー情報登録 |
